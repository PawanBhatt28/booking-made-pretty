# Booking Made Pretty

A restaurant reservation system built with React (Vite) and Spring Boot.

## Features
- **Frontend**: React, Vite, CSS Animations
- **Backend**: Spring Boot, H2 Database, Jakarta Validation
- **Integration**: REST API with CORS enabled

## Setup
1. **Backend**:
   ```bash
   cd backend
   mvn spring-boot:run
   ```
2. **Frontend**:
   ```bash
   npm install
   npm run dev
   ```
